# How to Play

Learn how to play Sudoku Logic Lab and master the game features.

## Game Basics

### What is Sudoku?

Sudoku is a logic-based number puzzle where you fill a 9√ó9 grid with digits 1-9, following these rules:
- Each **row** must contain the digits 1-9 exactly once
- Each **column** must contain the digits 1-9 exactly once
- Each **3√ó3 box** must contain the digits 1-9 exactly once

### Objective

Fill the entire grid correctly using logic and deduction. No guessing required!

## Getting Started

### 1. Select Difficulty

Choose from four difficulty levels:
- **Easy**: 40-45 clues, good for beginners
- **Medium**: 30-35 clues, moderate challenge
- **Hard**: 25-28 clues, requires advanced techniques
- **Daily**: Special puzzle that changes every day

### 2. Start Playing

Click a cell to select it, then:
- Type a number (1-9) to fill it
- Press backspace/delete to clear it
- Click another cell to move

### 3. Track Progress

Watch your stats in real-time:
- **Timer**: How long you've been solving
- **Moves**: Number of cells you've filled
- **Errors**: Invalid moves (if validation is on)

### 4. Complete the Puzzle

Fill all cells correctly to win. The game will:
- Validate your solution
- Stop the timer
- Show completion message
- Prompt you to save your score

## Tips for Beginners

### 1. Start with Easy Numbers
Look for rows, columns, or boxes with many clues‚Äîthey're easier to complete.

### 2. Use the Process of Elimination
If 8 numbers are already in a row, the missing one is obvious!

### 3. Look for Single Candidates
If only one number can go in a cell, fill it immediately.

### 4. Check Box-Row/Column Interactions
If a number must be in one row of a box, it can't be in that row elsewhere.

### 5. Take Breaks
Stuck? Step away for a few minutes. Fresh eyes help!

## Advanced Techniques

### Naked Pairs
If two cells in a row/column/box can only contain the same two numbers, eliminate those numbers from other cells.

### Hidden Pairs
If two numbers can only go in two cells in a row/column/box, those cells can't contain other numbers.

### X-Wing
If a number appears in only two positions in two rows, and those positions are in the same columns, eliminate that number from those columns.

### Swordfish
An advanced extension of X-Wing using three rows/columns.

### More Strategies
Practice these techniques in the game to improve your solving skills. The more you play, the better you'll get at recognizing patterns!

## Scoring System

### Time-Based Scoring
Your score is based on completion time:
- **Faster = Better**: Lower time = higher rank
- **Difficulty Matters**: Harder puzzles = separate leaderboards
- **Moves Don't Count**: Only time affects your score
- **Errors Don't Penalize**: But they slow you down!

### Leaderboard Ranking
Rankings are separate for each difficulty:
1. Easy Leaderboard
2. Medium Leaderboard
3. Hard Leaderboard
4. Daily Challenge Leaderboard

## Common Questions

### Can I pause the timer?
No, the timer runs continuously. This ensures fair competition on the leaderboard.

### What if I make a mistake?
Just clear the cell and try again. There's no penalty except the time spent.

### Can I play offline?
Yes! Puzzle generation works offline. Leaderboard and chat require internet.

### Do I need an account?
No! Play anonymously. Just enter a name when saving scores.

### Can I play the same puzzle again?
Not directly, but you can try the Daily Challenge or generate a new puzzle of the same difficulty.
