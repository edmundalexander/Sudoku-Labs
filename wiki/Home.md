## üìñ What is Sudoku Logic Lab?

Sudoku Logic Lab is a modern, browser-based Sudoku game featuring:
- Multiple difficulty levels (Easy, Medium, Hard, Daily)
- Global leaderboard system
- Real-time chat for players
- Campaign mode with progressive challenges
- Serverless architecture using GitHub Pages and Google Apps Script

## ü§ù Contributing to the Wiki

The Wiki is maintained alongside the codebase. To update:

1. Edit markdown files in the `/wiki` directory of the repository
2. Follow the guidelines in `WIKI_SETUP.md` in the repository root
3. Submit a pull request with your changes
4. Wiki pages will be synced to GitHub Wiki
